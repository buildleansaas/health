# Morning Check-In Questions (Pepper)

Use this in Discord. Keep one readable message, no shorthand codes.

## Prompt
- Wake window hit? (yes/no)
- Sleep quality (1-5)
- Morning energy (1-5)
- Guardrails today: caffeine cutoff, last meal cutoff, wind-down start
- Nutrition plan today (short)
- Training type today: Masters Swim / Gym LP / Home Strength / HIIT / Minimum Day / Recovery
- Fallback plan if schedule breaks: A (25-35m), B (10-15m), or C (8-12m)
- Hydration plan (1L by noon: yes/no + plan)
- Stress now (1-5) + reset time
- Meaningful connection planned (yes/no + who)
- Pain (0-10 + where)
- Any red-flag symptom (yes/no)
- Biggest risk + if-then response
- One non-negotiable
- Optional: awakenings, mood, schedule notes

After Austin replies:
- Pepper returns a short coaching read + top 1-3 actions.
- Pepper writes/updates `journals/YYYY-MM-DD-morning.md` and commits.
