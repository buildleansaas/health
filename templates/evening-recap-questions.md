# Evening Recap Questions (Pepper)

Use this in Discord. Required-first flow should be answerable in <90 seconds.

## Quick Reply Format
```text
Scores S/N/T/Total:3/2/2/7 | Train:HomeStrength fallback:A | Readiness:Yellow | StressPeak:4 reset Y | Hydration:Y | Connect:N | PainPeak:3 shoulder | RedFlag:N | Happened:swim,late call | Worked:prep lunch | Friction:late screen | Change:phone off 9:30
Optional: alcohol/cannabis, extra context
```

## Required first
1. "Evening recap (<90 sec) for YYYY-MM-DD."
2. "Give scores: sleep (0-4), nutrition (0-3), training (0-3), total (0-10)."
3. "Give training completion context: training mode token (`Train:GymLP|Train:HomeStrength|Train:HIITOnly|Train:MinimumDay|Train:Recovery`) + completion type (planned / fallback `A/B/C` / recovery)."
4. "Give modifier checks: readiness trend (Green/Yellow/Red), peak stress (1-5) + reset done (Y/N), hydration sufficiency (urine mostly pale + 1L by noon Y/N), connection 10+ min (Y/N)."
5. "Give safety check: peak pain (0-10) + location + red-flag symptom (Y/N)."
6. "Give reflection: what happened (2 short bullets), what worked (1), friction (1), one change for tomorrow."
7. "Optional: alcohol/cannabis, extra context."

If anything is missing, ask only for the missing required item.
